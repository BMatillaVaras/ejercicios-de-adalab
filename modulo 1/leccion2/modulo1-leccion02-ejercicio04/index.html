Lenguajes de programación front-end

El desarrollador Front-end trabaja del lado del Cliente, en el navegador, en el lado de lo que se ve. Principalmente se ocupa de los componentes externos del sitio web o de la aplicación web. Como consecuencia, deben cominar obligatoriamente: HTML, CSS y JavaScript.

HTML es un lenguaje de marcado, es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen información adicional acerca de su estructura: si es un título, un enlace o un párrafo.

CSSes un lenguaje de estilos para definir la presentación visual de un documento escrito en un lenguaje de marcado, como HTML.

JavaScript, a diferencia de HTML y CSS, sí que es un lenguaje de programación. Este lenguaje nos permite dar instrucciones al ordenador, en este caso al navegador web, para explicarle cómo debe mostrar nuestra página y qué debe hacer en qué situación (si se pulsa un botón, si se rellena un campo, si pulsamos un enlace, etc.)

También existen otros lenguajes y frameworks que se utilizan en front como pueden ser:

  1. Bootstrap
  2. Angular
  3. React
  4. Vue
